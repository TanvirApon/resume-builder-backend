# Resume Builder Backend

Backend for building and managing resumes with **Spring Boot** and **MongoDB**.  
Features secure REST APIs, email verification, PDF generation, and payment integration.

---

## Features
- User registration & login with JWT authentication  
- Email verification  
- Create, update, delete resumes  
- Generate PDF resumes  
- Razorpay payment integration  
- Cloudinary file uploads  

---

## Tech Stack
- **Spring Boot** | **MongoDB** | **JWT** | **Razorpay** | **Cloudinary**
